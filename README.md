## Modules Installation
- use git:
```bash
 git clone https://github.com/ajisaputra14/tea-tx
```
- Open terminal and make sure you already in bot folder:
```bash
cd tea-autosend
```
- Install modules:
```bash
npm install
```
- Run bot:
```bash
node index.js
```
- Then insert your EVM Wallet PrivateKey
